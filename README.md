Техническое задание: Приют для животных 
Платформа: ASP.NET MVC (.NET Framework / .NET Core)
СУБД: SQLite / JSON 

1. Цель проекта
Разработка веб-приложения для управления каталогом животных в приюте. Система должна обеспечивать просмотр, добавление, редактирование и удаление записей, а также разграничивать права доступа для обычных посетителей и администраторов.

2. Основной функционал (Страницы)
2.1. Главная страница  
Приветственный баннер с логотипом приюта. 

Ссылки на основной список и информацию о приюте. 

2.2. Список питомцев  
Таблица / сетка с краткой информацией о животных (фото, кличка, возраст, категория).

Фильтры: по виду, полу, возрасту, статусу.

Кнопка «Добавить питомца» (видна только авторизованным ролям Admin).

2.3. Подробности о питомце  
Детальная карточка: увеличенное фото, порода, описание характера, история, дата поступления, особые приметы.

Контактная информация для желающих забрать (телефон приюта, email).

2.4. Добавление питомца
Форма, включающая поля:

Кличка 
Вид 
Порода 
Возраст 
Описание 
Загрузка фото 

2.5. Редактирование питомца
Предзаполненная форма.

Возможность смены фото.

Кнопки «Сохранить» и «Удалить».

2.6. Справка о разработчике 
Информация о разработчиках (студенты, группа, ВУЗ).

Контактные данные для обратной связи.

Список используемых технологий.

3. Пользователи и разграничение доступа
Реализовать систему ролей.

Назначение роли Admin происходит вручную через базу данных.

4. Технический стек и хранение данных
Backend: C#, ASP.NET MVC (Core 8.0).

База данных:

Основной серверный режим: JSON?.

Примечание: Предусмотреть архитектуру репозитория, позволяющую в будущем переключиться на хранение в JSON-файлах, реализовав соответствующий IAnimalRepository.

Frontend: Razor Views, Bootstrap 5, TailWind.

5. Дополнительные требования
Валидация: Все формы должны иметь проверку на обязательность полей и формат данных.
Git: Код должен быть чистым, с осмысленными именами переменных и комментариями.
